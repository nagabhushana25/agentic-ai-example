# Reporting Layers

The reporting system has multiple layers:

Raw Layer:
- stores source-aligned extracted data
- minimal transformations
- used for auditing and replay

Staging Layer:
- standardized columns and formats
- used for validation and cleansing

Curated Layer:
- business-ready transformed data
- used by reporting and analytics teams

Presentation Layer:
- dashboard-specific aggregated tables
- optimized for BI tools and executive reporting

Typical reports:
- sales performance dashboard
- customer health dashboard
- support operations dashboard
- ETL monitoring dashboard
- backlog tracking dashboard
- revenue reconciliation report
