# Assignment Requirements

Create a production-ready multi-agent system using LangChain, LangGraph, and middleware concepts.

Required characteristics:

- Multi-agent architecture with an orchestrator and specialized sub-agents
- User-scoped memory management with per-user persistence
- Skills library for reusable procedures
- Middleware stack for summarization, tool-call limiting, PII protection, todo tracking, and retry logic
- Guardrails for input and output validation
- Vector search and Databricks integration
- Session management and conversation history
